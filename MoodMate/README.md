# 🔱MoodMate - My first python Mood & Habit Tracker

Hi there!
This is **ModeMate**, a begineer- friendly project i created using python.
It's a simple app that checks in on how you're feeling and whether you've done some healthy habits for the day.

---

## What It Does

- Ask for your name and how you're feeling ( your mood)
- checks if you:
  sleep well
  eat breakfast first before water
  exercise
- gives you a short, uplifting message based on your answers
  saves in a log file if you want to keep a record

--- What I Used -**python 3.13** - for all the code -**VS code** - to write and run code -**Terminal** - to test it

- simple tools like:'input()', 'if/else','bool', 'datetime',and debugging
